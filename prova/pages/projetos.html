Tabelas

Desenvolvi uma página web utilizando HTML para criar uma tabela organizada e semântica, com foco em exibir dados de forma clara e acessível. O projeto foi codificado no Visual Studio Code e demonstra conhecimentos em estruturação de conteúdo com HTML, além de boas práticas como o uso adequado de tags como table, thead, tbody e tfoot. Ideal para representar dados em relatórios, listas ou catálogos simples

Artistas

Criei uma galeria interativa utilizando HTML no Visual Studio Code, destacando as melhores artistas femininas. Cada artista é representada por uma imagem; ao clicar nela, o usuário é redirecionado para uma página individual dedicada à artista, também desenvolvida por mim. O projeto demonstra domínio na estruturação de páginas com HTML, uso de links e imagens, além de uma navegação simples e funcional entre páginas. É um exemplo de como criar uma experiência de navegação em múltiplas páginas estáticas.

Comandos básicos

Desenvolvi uma página informativa em HTML no Visual Studio Code com o objetivo de apresentar os principais comandos e atalhos do editor. O projeto inclui uma tabela organizada com descrições claras de cada comando, facilitando o aprendizado de iniciantes. Além disso, a estrutura simples e o conteúdo direto tornam o projeto útil como material de apoio ou referência rápida para quem está começando a programar com o VSCode.
